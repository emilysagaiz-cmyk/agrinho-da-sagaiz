# Comedouro Sustentável

Landing page inspirada em páginas do setor agro para divulgação de comedouros e equipamentos rurais.

## Estrutura

- index.html
- style.css
- script.js
- README.md

## Funcionalidades

- Hero banner
- Seção do produto
- Benefícios
- Galeria
- Formulário de contato
- Layout responsivo

## Como usar

1. Baixe os arquivos.
2. Mantenha todos na mesma pasta.
3. Abra o arquivo `index.html` no navegador.

## Tecnologias

- HTML5
- CSS3
- JavaScript
